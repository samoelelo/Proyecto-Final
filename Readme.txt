Librerías necesarias para correr el programa.

*numpy

	Desde la ventana de comandos (ya sea windows o linux) escribir:

	pip install numpy

*math
	Desde la ventana de comandos (ya sea windows o linux) escribir:

	pip install python-math

*nii babel -librería para leer neuroimágenes
	Requerimientos:
		Python 3.6 or greater

		NumPy 1.14 or greater

		Packaging 14.3 or greater

	Instalación:
		Windows o OSX
		Escribir en terminal:
		pip install nibabel

		Ubuntu/Debian:
		apt-get update
		apt-get install python-nibabel
